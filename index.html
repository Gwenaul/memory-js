Le Jeu Memory !

Le jeu Memory est un jeu de mémoire, trouvez la bonne paire !

Le but est de retrouver les paires en ne retournant que deux cartes à la fois. Le joueur retourne deux cartes de son choix. S'il obtient les mêmes motifs, il remporte la paire puis rejoue. S'il se trompe, il retourne les deux cartes faces cachées exactement au même endroit et rejoue. La partie est terminée lorsque toutes les cartes ont été assemblées par paires.
